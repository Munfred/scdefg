Differential expression on C. elegans single cell RNA sequencing data

This app allows you to quickly perform differential expression on single cell RNA sequencing data. Just select cell types and experiments to compare and some genes to highlight. It will produce an interactive volcano plot and table with results in ~15s.

This app uses scvi-tools. The code for the app is available on GitHub.

To highlight genes add one gene per line below. Gene names (bus-1) or ID are accepted (WBGene00018223)
