Differential expression on CeNGEN C. elegans single cell RNA sequencing data

It allows you to quickly perform differential expression on single cell RNA sequencing data from the CeNGEN project 2020 data release, described in the preprint Molecular topography of an entire nervous system. Just select cell types and experiments to compare and some genes to highlight. It will produce an interactive volcano plot and table with DE results in ~15s.

This app uses scvi-tools. The codefor the app is available on GitHub.

To highlight genes add one gene per line below. Gene names (bus-1) or ID are accepted (WBGene00018223)
